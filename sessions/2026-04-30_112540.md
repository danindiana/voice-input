## Changes Made

### 1. Ambient transcript auto-save (`voice-ambient/src/main.rs`)

**Problem:** Ambient mode transcripts were ephemeral — displayed in the TUI only, lost on exit. The optional `--db` SQLite flag existed but nothing was saved by default.

**Fix:** Each ambient session now auto-creates a timestamped plain-text file:
```
~/.local/share/voice-input/transcripts/YYYY-MM-DD_HH-MM-SS.txt
```
Each utterance is appended as `[HH:MM:SS] text`. The TUI footer gained a `SAVE: <filename>` indicator (green when saving, dark-gray when `SAVE: off`).

Pass `--no-save` to opt out. The directory is created automatically via `fs::create_dir_all`.

**Implementation notes:**
- `transcript_path: Option<String>` added to `App` struct
- File opened in `spawn_reader()` using `OpenOptions::new().create(true).append(true)` — matches the existing SQLite failure-tolerance pattern (`and_then(...).ok()`)
- `chrono::Local::now()` used for the filename timestamp (already a dep)
- `use std::io::Write` added for `writeln!`

### 2. `--submit` flag (`voice-input.sh`)

**Problem:** STATUS.md TODO — auto-submit mode for use with Claude Code and terminal prompts.

**Fix:** `voice-input --type --submit` sends `xdotool key Return` after typing.

### 3. Configurable whisper model

**Problem:** Model was hardcoded to `medium` in both `ambient.py` and `transcribe.py`.

**Fix (phase 1):** Added `VOICE_WHISPER_MODEL` env var support (default: `medium`) to both scripts.

**Fix (phase 2):** Added `--model <name>` flag to `voice-input.sh`. Sets `VOICE_WHISPER_MODEL` via `export` before exec/subprocess calls — inherited by both `ambient.py` (via `exec` env) and `transcribe.py` (via subprocess env).

**Fix (phase 3):** After testing `large-v3` confirmed working, default bumped from `medium` → `large-v3` in both Python scripts.

Usage:
```bash
voice-input --ambient                    # large-v3 (new default)
voice-input --ambient --model medium     # opt back to medium (faster/lighter)
voice-input --model large-v3 --print     # push-to-talk with large-v3
```

### 4. Project logo added

`voice-input-logo.png` added to repo root and embedded at the top of `README.md`.
